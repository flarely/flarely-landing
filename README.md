# Flarely Landing Page

Marketing landing page for Flarely — webhook-based error notifications.

Built with Next.js, Tailwind CSS, and deployed on Vercel.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view.

## Deploy

Connect this repo to Vercel for automatic deployments:
1. Push code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Import Project"
4. Select this repository
5. Deploy

Environment variables (if needed):
- `NEXT_PUBLIC_APP_URL` — your app URL (default: https://flarely-cloud.fly.dev)
